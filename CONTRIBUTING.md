# Contributing

Thanks for contributing! Please follow these simple rules:

- Fork the repository and create a feature branch: `feature/your-feature`.
- Open a Pull Request against `main`.
- Add a short description and link any related issue.
- Request at least one review before merging.
- Write clear commit messages and keep commits focused.
- Run tests and linting before opening PR.

If you're unsure about something, ask in an issue or contact the Project Manager.
